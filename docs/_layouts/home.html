Falcon

A Performant and Simple Banking Backend

Falcon is a simple bank management system that works via a REST API.

Features

Create Accounts

Choose between various different account types - savings, current, and more.

Run Transactions

Create transactions between accounts - no matter the type.

Built for High Seas 2024/2025

Falcon was built for the High Seas 2024/2025 event, as my first project in C++.
